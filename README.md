# parametric-03.iverfinne.no

A parametric 3D studio. The UI shell (viewer stage, controls panel,
gestures, shareable URL state) comes from parametric-01.iverfinne.no;
the generative motor is its own — a totem system of stacked, pierced,
limb-sprouting bodies in hammered bronze. See lib/engine.ts (parameter
space) and lib/totem.ts (SDF + marching cubes geometry).
